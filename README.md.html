Linode v4 API

This is a promise-based clinet for the Linode API version 4

Installation

npm install linode-v4 --save

Usage

import Linode from 'linode-v4';

 

const api = new Linode(api_key);

or

const Linode = require('linode-v4').default;

 

const api = new Linode(api_key);
